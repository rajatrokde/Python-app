# 📝 Flask ToDo App

A simple web-based ToDo list app built with Python and Flask.

---

## 🚀 Features

- Add tasks via a web form
- View all tasks
- Mark tasks as done
- Delete tasks
- In-memory storage (resets on server restart)

---

## 🛠️ Requirements

- Python 3.6+
- Flask

Install Flask using pip:

```bash
pip install flask
